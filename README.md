# AI Remote Job Finder

An AI-powered job discovery and resume matching system built using n8n, Ollama, and Qwen.

## Features

- Fetches remote jobs from public APIs
- Filters non-English language jobs
- Filters senior and experience-heavy roles
- Identifies fresher-friendly opportunities
- Scores job relevance using AI
- Supports AI/ML, Data Analyst, Python, Automation, and Agentic AI roles
- Designed for Indian freshers seeking remote opportunities

## Tech Stack

- n8n
- Ollama
- Qwen3
- JavaScript
- REST APIs

## Workflow

1. Fetch jobs from API
2. Filter unsuitable jobs
3. Analyze jobs using AI
4. Score job matches
5. Recommend applications
